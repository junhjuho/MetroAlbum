## 🚇 지하철 인증샷 AR (가제)

### 프로젝트 일정
- 2025-11-05 ~ 11-18

### 프로젝트 당담
- 장주호

### 프로젝트 기술
- Unity 2022.
- AR Location (ARFoundation + GPS)
- Firebase

### 프로젝트 개요

핵심 기능: GPS 기반으로 지하철역 반경 진입 시 자동 인식 → 역명/노선 표시 + 인증샷 촬영 기능 제공.

모드 구성: (1) 촬영 모드 (2) 보기 모드 (3) 내 설정 — 사진 업로드, 좋아요·신고, 이동 횟수 통계 포함.

위치 인식: 백그라운드에서도 지속 체크, 중복 방지 규칙(다른 역 이동 or 1시간 후 재인식).

확장 기능: 광고 구독제(500~1000원), 웹뷰 갤러리, 향후 노선도/공구/다국어 지원.

개발 포인트: GPS·카메라·웹뷰 연동, 서버 저장 및 신고 처리 로직, 자동 역 인식 알고리즘 설계.
